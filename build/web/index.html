New Beginnings

I had thought one year ago when I first made a blog that it would really stick and be all the things I wanted, but time got away from me and fell into ruin. This time around, instead of hosting my own blog, I thought I'd just give a link from my page to the good folks at Blogger.

I'm a freshly graduated Computer Scientist with a strong passion for games and game development. A brief introduction can be found here: About.

My plan for this blog is to make games and document that process here. While I have a few ideas in the works with my friend Phil Kovac, I haven't started the coding process to really put down anything. I hope whoever does read this blog, can learn from my learning process as I take on these big coding and game design projects. Thank you for reading and I hope you enjoy the ride!

~Andrew Wilkinson

Current Project

The project I'm currently working on at the University of Massachusetts Lowell is Weave, a Web-based Analysis and Visualization Environment.

My role in this project has been one of bug fixing and some feature implementation. The main feature I'm working on right now is Collaborative Analysis, or the ability for multiple people to work together to analyze data sets.
